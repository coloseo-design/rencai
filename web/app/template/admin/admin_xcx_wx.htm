APPID:
绑定支付的APPID（必须配置，公众号平台可查看）
AppSecret:
绑定支付的公众号AppSecret（必须配置，公众号平台可查看）
商户号(MCHID):
商户号（必须配置，开户邮件中可查看）
商户密钥(KEY):
商户支付密钥，参考开户邮件设置（必须配置，登录商户平台自行设置）
证书（pem格式）路径:
微信商户平台-API安全-下载证书，证书不要存放于网站目录下，防止被恶意下载，请填绝对路径 如D:\wxca\apiclient_cert.pem
证书密钥（pem格式）路径:
微信商户平台-API安全-下载证书，证书不要存放于网站目录下，防止被恶意下载，请填绝对路径 如D:\wxca\apiclient_key.pem
xcxAPPID:
绑定支付的小程序APPID（必须配置，小程序平台可查看）
xcxAppSecret:
绑定支付的小程序AppSecret（必须配置，小程序平台可查看）
原始ID:
微信小程序原始ID（使用视频面试功能必须配置，小程序平台-设置-基本设置中可查看）
