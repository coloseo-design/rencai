配置说明：助力权益按照梯次分配，每次好友助力达到设置的数量可以获取对应的权益
举例说明：
刷新职位1次，达标助力10人 : 代表 当助力人数到达10人时，企业可以获得一次职位刷新的权限.
下载简历1份，达标助力50人：当好友继续助力，人数达到50人，在原来已有的职位刷新1次权益基础上又新获得下载简历1份的权益。
以此类推，可以多项权益设置相同助力人数，达到助力指标则同时获得多项权益，任务结束按照助力总人数进行统计发放相关权益。
Ps：权益、达标助力参数设置为0或空，则该项权益不参与好友助力任务。
