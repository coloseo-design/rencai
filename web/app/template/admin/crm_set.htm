周期设置：
  • 跟进提醒 天
  • 释放周期 天
  • 成交限定 天
  • 领取限定 天
1-跟进提醒：X天，超过X天未跟进，自动提醒跟进
2-释放周期：X天，超过X天未跟进，自动释放至资源池（释放周期 大于 跟进提醒
3-成交限定：X天，已领取为起始时间，当前会员等级在规定周期内未发生改变，自动释放至资源池
4-领取限定：X天，资源释放后，上一任CRM在规定周期内禁止重复领取
0 天 表示不限制
